# R223W teacher draft primary reading shell spec

## Policy

Teacher draft must be the primary reading object in any review-only pilot gate fixture.

## Source

The shell must load or link canonical teacher manuscripts:

- `R223M_P4_P1_teacher_readable_process_v6.html`
- `R223N_P3_P1_teacher_manuscript_draft_v5.html`
- `R223O_P1_teacher_manuscript_draft_v2.html`

## Display rule

The pilot gate shell may:

- link to canonical teacher manuscripts
- embed readonly excerpts only if clearly sourced
- show a document-style reading shell

The pilot gate shell must not:

- regenerate teacher manuscript text
- rewrite teacher manuscript body
- create a new R223W teacher draft
- display ledger fields inside the teacher draft

## Primary reading order

```text
safety banner
canonical source label
teacher manuscript reading area
difference summary
collapsed ledger summary
review action controls
```

## Hold condition

Hold if the teacher manuscript becomes secondary to ledger, schema or component metadata.
